========================
 파이썬 디자인 패턴
========================

환영합니다!

저는 **Brandon Rhodes**입니다
(`웹사이트 <https://rhodesmill.org/brandon/>`_,
`트위터 <https://twitter.com/brandon_rhodes>`_).
이것은 `파이썬 프로그래밍 언어 <https://www.python.org/>`_의
디자인 패턴에 대한 저의 발전적인 가이드입니다.

* 이 사이트를 통해 파이썬과 디자인 패턴에 대한 제 아이디어를
  한곳에 모을 수 있게 되었습니다.

* 이 페이지들이 웹 검색에서 더 쉽게 찾을 수 있고,
  제 `파이썬 컨퍼런스 강연 <http://rhodesmill.org/brandon/talks/>`_의
  비디오와 슬라이드에 흩어져 있을 때보다 더 쉽게 읽을 수 있도록 하여
  패턴을 더 발견하기 쉽게 만들기를 바랍니다.

* 다른 의무들의 무게로 인해 제 진행은 간헐적입니다.
  새로운 자료를 확인하려면,
  이 사이트의 `GitHub 프로젝트 저장소 <https://github.com/brandon-rhodes/python-patterns>`_의
  커밋 기록을 방문하시면 됩니다.
  거기서 "Watch"를 선택하여 업데이트를 받을 수도 있습니다.

이러한 사전 설명이 완료되었으니, 이제 패턴들을 소개합니다!
